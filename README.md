# Academic Portfolio - Dr. Shafaque Zehra

A fast, responsive, SEO-friendly static academic portfolio. 

## Features
* Light/Dark mode toggle
* Real-time publication filtering
* Contact form via Formspree
* Semantic HTML5 & CSS3 with zero heavy dependencies

## How to Deploy to GitHub Pages
1. Create a new repository on GitHub named `yourusername.github.io`.
2. Upload all the files (`index.html`, `style.css`, `script.js`, etc.) to the `main` branch.
3. Replace `https://yourusername.github.io` in the `index.html` (JSON-LD) and `sitemap.xml` with your actual URL.
4. Set up the contact form by replacing `YOUR_FORMSPREE_ID` in `index.html` with an endpoint from [Formspree.io](https://formspree.io/).
5. Upload your CV as a PDF file named `cv.pdf` into the same root folder.
6. Go to **Settings > Pages** in your GitHub repository, set the source to `main` branch, and click Save. Your site will be live in a few minutes!
